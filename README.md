## 📂 Completed Projects

### 1️⃣ Task 1: Universal Language Translation Tool
A real-time translation application with a user-friendly GUI. It allows users to input text and instantly translate it into over 100 supported languages.
* **Tech Stack:** Python, Gradio, `deep-translator` (Google Translate API)
* **Features:** Auto-language detection, smooth UI, instant translation.

### 2️⃣ Task 2: Smart FAQ Chatbot (NLP based)
An intelligent customer support chatbot designed to answer Frequently Asked Questions. Instead of basic rule-based (if-else) logic, this bot uses vector embeddings to understand the *meaning* of the user's question and finds the most relevant answer.
* **Tech Stack:** Python, Gradio, `sentence-transformers` (HuggingFace), `scikit-learn`, Pandas.
* **Features:** Cosine similarity for intent matching, Small-talk engine (greetings/bot identity), high-accuracy semantic search.

### 3️⃣ Task 4: Real-Time Object Detection & Tracking
A powerful Computer Vision application that detects objects in real-time. Users can upload an image or use their webcam to instantly identify and classify multiple objects in the frame with bounding boxes and confidence scores.
* **Tech Stack:** Python, Gradio, Ultralytics (`YOLOv8`), OpenCV.
* **Features:** Fast inference using YOLOv8 Nano, bounding box rendering, live webcam support.
